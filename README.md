## A dissertation project investigating the threat of AI-generated phishing emails and its mitigation

## Overview

This project explores whether modern NLP-based classifiers can detect phishing
emails generated by large language models (LLMs) such as DeepSeek. It compares
a TF-IDF + Logistic Regression baseline against a Bidirectional LSTM (BiLSTM)
deep learning model, evaluating both on a realistic test set containing
AI-generated phishing emails unseen during training.

## Requirements

- Python 3.9+
- pandas
- scikit-learn
- TensorFlow 2.x

Install dependencies with:

bash
pip install pandas scikit-learn tensorflow


## Usage

Run the scripts in order:

bash
python src/1_build_dataset.py
python src/2_preprocess.py
python src/2b_clean_dataset.py
python src/3_vectorise.py
python src/4_baseline_model.py
python src/5_feature_experiments.py
python src/6_deep_model.py
python src/7_deep_model_cross.py
python src/8_ablation_study.py
python src/9_cross_model_eval.py
